Jakou strukturu webu zvolíme?

Tři přístupy s různým rozsahem a zaměřením

A

Minimální marketingový web (3–4 stránky)

Stránky: Domů · Jak pomoci · O kempu · Kontakt

CMS: pouze blog/aktuality a loga sponzorů

Výhody

  • Rychlá realizace
  • Snadno se udržuje
  • Soustředěný na konverzi

Nevýhody

  • Méně prostoru pro příběhy
  • Těžko se rozšiřuje
B

⭐ Plnohodnotný web (5–6 stránek) — doporučeno

Stránky: Domů · O kempu (historie + milníky) · Jak pomoci (3 cesty: dárce, sponzor, dobrovolník) · Galerie & Aktuality · Partneři & Transparentnost · Kontakt

CMS: aktuality, fotogalerie, sponzoři, milníky, příběhy

Výhody

  • Pokrývá všechny cílové skupiny
  • Buduje důvěru (20 let, transparentní účet)
  • Báře dává smysluplný obsah ke správě
  • Vhodné pro Rotary magazín

Nevýhody

  • Více práce na začátku
  • Vyžaduje dobrý obsah od Báry
C

Jednostránkový landing page (scroll)

Struktura: Hero → Příběh → Data → Jak pomoci → Galerie → Sponzoři → Kontakt (vše na jedné stránce)

CMS: minimální — jen galerie a novinky

Výhody

  • Moderní, velmi soustředěný
  • Skvělý pro konverzi
  • Rychlé na scrollování

Nevýhody

  • Těžko linkovat konkrétní sekce
  • Méně hloubky pro sponzory
  • Galerie a aktuality jsou ošemetné

Doporučení: Varianta B nejlépe odpovídá potřebám — oslovuje Rotary kluby, firemní sponzory i individuální dárce, zároveň poskytuje prostor pro příběhy a budování důvěry, které jsou klíčem k získání stabilních partnerů.
